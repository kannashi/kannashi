linux上mysql5.7安装

发表于 2020-04-16 | 分类于 运维
字数统计: 421words | 阅读时长 ≈ 2min

1. 下载mysql安装包

地址：https://dev.mysql.com/downloads/mysql/

2. 使用ftp工具上传到/usr/local

3. 解压安装包:

tar -zxvf mysql-5.7.27-linux-glibc2.12-x86_64.tar.gz

4. 修改文件夹名称：

mv /usr/local/mysql-5.7.27-linux-glibc2.12-x86_64 /usr/local/mysql-5.7.27

5. 创建mysql用户组和用户


groupadd mysql

useradd -r -g mysql -s /bin/false mysql

6. mysql目录授权给mysql组和mysql用户


chown mysql:mysql mysql

chmod 750 mysql

7. 初始化数据库

记下最后打印的临时密码

8. 配置my.cnf

如果/etc目录下存在my.cnf则直接编辑

vim /etc/my.cnf

如果/etc目录下不存在my.cnf，则先创建文件，再编辑


mkdir /etc/my.cnf

vim /etc/my.cnf

我配置的编码格式是uft8mb4，配置文件内容如下：


[mysqld]

character-set-client-handshake = FALSE

character_set_server=utf8mb4

collation-server = utf8mb4_unicode_ci

init_connect='SET NAMES utf8mb4'

basedir=/usr/local/mysql-5.7.27

datadir=/usr/local/mysql-5.7.27/data

socket=/tmp/mysql.sock

log-error=/var/log/mysqld.log

pid-file=/var/run/mysqld/mysqld.pid
[mysql]
default-character-set=utf8mb4
[client]
default-character-set=utf8mb4

9. 配置开机启动

cp support-files/mysql.server /etc/init.d/mysqld

10. 环境变量设置

编辑profile文件

vim /etc/profile

添加如下设置到最后一行

export PATH=$PATH:/usr/local/mysql-5.7.27/bin

保存配置文件,使其生效

source /etc/profile

11. 修改密码

首先启动mysql

service mysqld start

登录mysql

mysql -u root -p

输入之前存的临时密码，进入mysql命令行，设置密码为123456，执行


mysql>set password=password('123456');

mysql>grant all privileges on *.* to root@'%' identified by '123456';

mysql>flush privileges;

退出

mysql>exit;

完成配置。
